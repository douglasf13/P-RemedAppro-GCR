# P-RemedAppro-GCR

- Comment utiliser

Pour tourner l'application, il faut télécharger déjà l'executable et pour l'tuliser vous devez avoir soit le fichier .csv fourni, soit avoir un fichier à vous nommé
"musiques.csv" avec la structure suivante:

Title;Artist;Genre;Year;Language;Idioma;Langue;Mood;Humor;Humour;Album;

dans le meme dossier. 

- Remarques

Eviter d'utiliser des accents, l'encodage n'est pas bien defini, du coup les accents peuvent empecher d'avoir le résultat desiré.
Toutes les entrées doivent etre mises exactement comme écrit dans les options, le program est encore sensible aux minuscules/majuscules.
